Welcome to the Cube Game homepage!

The Cube Game is a fun game where you simulate a spinning cube in a 3D void.
!! DISCLAIMER: The cube game contains NO FILTERS, you are responsible for what you see in the game. The Cube Game holds no ill intent but is open to everyone. We are not responsible for the actions of the public. !! Controls:
WASD - move horisontally
Left & Right arrows - turning around
Space - Move up
(Left) Shift - Move down
 To download the game, click here !

Installation guide


1. Download the file "cube game.zip". Download link located above.
2. Unzip the contents of the file to an empty folder anywhere on your computer. The recommended location is C:\Program Files
3. Open the file called "start" or "start.exe"
This will open a terminal, prompting you to type a username. Your name can be up to 20 characters.


You're all set! Enjoy the cube!
